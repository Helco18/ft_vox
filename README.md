# Stack technique

Ce projet utilise la bibliothèque [GLFW](https://www.glfw.org) pour la
création de la fenêtre et la gestion des entrées, ainsi que [Vulkan](https://www.vulkan.org) pour le rendu graphique.

# Lib utilisées

- [GLFW](https://www.glfw.org) : pour la création de la fenêtre et la gestion des entrées.
- [Vulkan](https://www.vulkan.org) : pour le rendu graphique.
- [GLM](https://github.com/g-truc/glm) : pour les mathématiques de l'API graphique.
- [stb_image](https://github.com/nothings/stb) : pour le chargement d'images.
